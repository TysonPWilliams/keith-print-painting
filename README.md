# Keith Print Painting & Decorating Website

A professional, responsive website for Keith Print Painting & Decorating, designed for brand awareness and lead generation.

## Features

- **Mobile-First Design**: Optimized for all devices with responsive layout
- **Professional Branding**: Clean, modern design that builds trust
- **Service Showcase**: Comprehensive display of painting and decorating services
- **Contact Integration**: Easy-to-use contact form and contact information
- **SEO Optimized**: Proper meta tags and semantic HTML structure
- **Accessibility**: WCAG compliant with proper focus states and screen reader support

## Sections

1. **Header**: Fixed navigation with company branding
2. **Hero Section**: Compelling call-to-action with service overview
3. **Services**: Six key service offerings with detailed descriptions
4. **About**: Company information and key differentiators
5. **Contact**: Contact form and business information
6. **Footer**: Additional links and company details

## Technologies Used

- **React 19**: Modern React with latest features
- **Vite**: Fast build tool and development server
- **CSS3**: Traditional CSS with modern features (Grid, Flexbox, Custom Properties)
- **Mobile-First**: Responsive design starting from mobile breakpoints

## Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd keith-print-painting
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

### Building for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## Customization

### Colors
The website uses a professional color palette:
- Primary: #e74c3c (Red)
- Secondary: #3498db (Blue)
- Dark: #2c3e50 (Dark Blue)
- Light: #f8f9fa (Light Gray)

### Content Updates
- Update company information in `src/App.jsx`
- Modify contact details and service descriptions
- Replace placeholder images with actual business photos
- Update phone numbers and email addresses

### Styling
- All styles are in `src/App.css` using traditional CSS
- Mobile-first responsive design with breakpoints at 768px, 1024px, and 1200px
- Easy to customize colors, fonts, and layout
